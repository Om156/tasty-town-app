# Tasty Town

A full-stack food ordering application with Spring Boot backend and React frontend.

## Project Structure

- `tasty town - backend/` - Spring Boot REST API backend
- `tasty town - frontend/` - React frontend application

## Features

- User authentication and authorization
- Food menu browsing and search
- Shopping cart functionality
- Order placement and tracking
- Admin panel for managing categories and food items
- Image upload and management

## Tech Stack

### Backend
- Java Spring Boot
- Spring Security with JWT
- MySQL Database
- Maven

### Frontend
- React
- Vite
- CSS3

## Getting Started

### Prerequisites
- Java JDK 17+
- Maven
- Node.js and npm
- MySQL Database

### Backend Setup
1. Navigate to `tasty town - backend/`
2. Configure database in `src/main/resources/application.properties`
3. Run `mvn spring-boot:run`

### Frontend Setup
1. Navigate to `tasty town - frontend/`
2. Run `npm install`
3. Run `npm run dev`
